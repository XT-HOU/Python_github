二、安装MATLAB

1、挂载镜像并开始运行 install 文件
cd ~                    # 切换到home目录
sudo mkdir matlab       # 创建一个文件夹，并命名为matlab
cd matlab                #切换到matlab目录
sudo mount -t auto -o loop Matlab98R2020a_Lin64.iso        # 挂载dvd，路径为iso镜像文件的路径
sudo ./install # 开始安装
2、选择“使用密钥安装”

①同意一些协之后输入安装Key:09806-07443-53955-64350-21751-41297


②提示选择许可证，解压Linux_Patch.7z文件,得到Crack文件夹,选择其中的license_standalone.lic许可证文件,点击下一步;选择安装位置,默认下一步即可;


③选择要安装的组件,自己选择,如果全选也可以,就是安装时间较长;点击下一步开始安装过程,大概半小时左右可以安装完成;(视磁盘性能而定)

3、取消挂载
sudo umount matlab/

到这里就安装完成了，接下来我们需要进行激活。

三、激活MATLAB

安装完成后,拷贝Crack文件夹中的libmwlmgrimpl.so文件到目录 /applications/Polyspace/r2020a/bin/maci64/matlab_startup_plugins/lmgrimpl/ 替换同名文件即可；

sudo cp Linux_Patch/Mathworks Matlab R2020a Linux x64 Crack/libmwlmgrimpl.so  /applications/Polyspace/r2020a/bin/maci64/matlab_startup_plugins/lmgrimpl

四、创建快捷启动方式

如果没有这一步操作，那么我们每次运行 matlab，都需要切换到安装路径的 bin 文件夹下面，然后执行 sudo ./matlab 才可运行 matlab，这样比较麻烦，下面我们在创建一个快捷启动方式以便于我们以后启动 matlab。
方法一：
1、创建一个桌面文件并输入下图中的内容

sudo vim /usr/share/applications/Matlab2020b.desktop

[Desktop Entry]
Categories=Development;Matlab;
Comment[zh_CN]=Matlab: The Language of Technical Computing
Comment=Matlab: The Language of Technical Computing
Exec=sh /usr/local/MATLAB/R2020b/bin/matlab -desktop
GenericName[zh_CN]=Matlab2020b
GenericName=Matlab2020b
Icon=/usr/local/MATLAB/R2020b/toolbox/sl3d/mainpage/matlab_logo.gif
Mimetype=
Name[zh_CN]=MATLAB
Name=MATLAB
Path=
StartupNotify=true
Terminal=false
Type=Application

方法二：

在Ubuntu软件里找到MATLAB安装，然后根据要求将安装路径里的./bin/matlab放入，便可将通过桌面图标打开Matlab
